## FAQ

**Q1: Does this require access to my company's billing accounts?**
No. The audit is entirely self-reported. You tell us which tools you use and what plans you're on. We compare that against our pricing database and find optimization opportunities. No API keys, no bank account connections, no security review needed.

**Q2: How accurate are the savings estimates?**
Our pricing data is manually verified against each tool's published pricing page every 2 weeks. We show exact dollar figures based on current listed prices. If a plan price changes, we update the database within 48 hours. Estimated savings are accurate to within ±5% of what you'd actually save by switching.

**Q3: Is my data stored or shared?**
We store audit results so you can access them via a shareable link. We never sell your data. Your email and company name are only used if you explicitly request a Credex consultation. You can delete your audit result at any time.

**Q4: What if my tool isn't in the database yet?**
We add new tools weekly based on user requests. Drop us a note through the "Request a tool" button and we'll add it within 7 days. Currently covering ChatGPT, Claude, GitHub Copilot, Cursor, Gemini, Windsurf, Midjourney, Perplexity, Notion AI, Grammarly, and 40+ others.

**Q5: I'm a solo founder. Is this useful for me?**
Yes. Solo founders are our fastest-growing segment. If you have 3+ AI subscriptions and aren't sure you need all of them, the audit will show you exactly where you're overspending. One user saved $600/year by dropping two redundant tools.
